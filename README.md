# 🕳️ corneHole - My custom ZMK config 🕳️

This is my custom ZMK Config for my corne v3 split-ergonomic 42-key board. I took inspiration from Miryoku and Sunaku. This config features Home Row Mods and a custom Vim layer for mimicing Vim's normal mode when editing text outside of the popular editor. The Vim layer is still a work in progress.

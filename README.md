# 🕳️ corneHole - My custom ZMK config 🕳️

This is my custom ZMK config for my split ergonomic 42-key [corne keyboard](https://github.com/foostan/crkbd). It uses [Miryoku](https://github.com/manna-harbour/miryoku) inspired layers with home-row mods, as well as a Vim layer meant to mimic vim's normal mode functionality when using text editors/fields without a native vim mode (i.e. MS Office, notepad, etc).

> [!WARNING]
>
> This is very much a work-in-progress with frequent layout changes.

![keymap-corne](keymap-drawer/corne.svg)

> Drawn with [@caksoylar's Keymap Drawer](https://github.com/caksoylar/keymap-drawer)

## Sources of Inspiration
- [Miryoku](https://github.com/manna-harbour/miryoku): Layers designed orthogonally with a single purpose per hand.
- [precondition's Home Row Mods](https://precondition.github.io/home-row-mods): Great suggestions for dialing in the home row mods just right. 
- [minusfive's Knucklehead](https://github.com/minusfive/knucklehead): Colorscheme for the keymap drawer
